# Notebook and Analysis Notes

This folder contains the project analysis documentation for the electricity demand forecasting system.

Steps to explore:
1. Use `src/tsa_project/data_generation.py` to create synthetic hierarchical demand data.
2. Use `src/tsa_project/forecasting.py` to train a global LSTM and create revenue forecasts.
3. Use `presentation/generate_ppt.py` to generate a short slide deck describing the architecture.
